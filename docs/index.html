Въведение

Това е бързо ръководство, което ще ви помогне да изградите проекта на ваша лична машина.

Настройка

Преди всичко, ще ви е нужен Node.js . Можете да го изтеглите оттук

Когато Node.js е инсталиран, отворете терминала си в папката на проекта и изпълнете командата npm installда се инсталират всички допълнителни модули. Ще се появи нова папка под името node_modules автоматично.

npm install

След като приключи изтеглянето и инсталирането на модулите сте готови. Сега трябва да изпълните командата npm start в терминала. Това ще събере всички файлове, които са му нужни и пусне уеб сървър на адрес http://localhost:80

npm start

Докато горната команда работи, тя ще поддържа сървъра, за да можете да виждате всякакви промени направени по файловете.

Използвайте комбинацията Ctrl+C или просто затворете терминала, за да спрете сървъра.
